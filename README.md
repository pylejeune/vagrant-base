vagrant-php-box
===============

Set up super fast a PHP5 development box with apache, mysql, phpmyadmin and whatever else that you may need


<h2>How to run</h2>

<ul>
  <li>
    Install vagrant using the installation instructions in the 
    <a href="http://docs.vagrantup.com/v2/getting-started/index.html" target="_blank">Getting Started document</a>
  </li>
  <li>Clone this repository, go to it, and run init.sh then run config.sh</li>
  <li>Put your sites and php files inside the sites folder</li>
  <li>Acess localhost:8080 or base.local</li>
</ul>


<h2>Included components</h2>

<ul>
  <li>php5</li>
  <li>apache2</li>
  <li>php5-cli</li>
  <li>php5-xdebug</li>
  <li>php5-mysql</li>
  <li>php5-imagick</li>
  <li>php5-mcrypt</li>
  <li>php-pear</li>
  <li>php5-dev</li>
  <li>php5-curl</li>
  <li>php5-sqlite</li>
  <li>libapache2-mod-php5</li>
</ul>
